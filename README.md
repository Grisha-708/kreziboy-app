# KREZIBOY — Android приложение (WebView)

Это готовый проект Android-приложения — обёртка сайта **kolosoww70.temp.swtest.ru**.
Приложение открывает сайт внутри себя, выглядит и работает как обычное приложение.

## Что умеет
- Открывает сайт на весь экран (без адресной строки браузера)
- Кнопка «Назад» листает историю сайта
- Свайп вниз — обновление страницы
- Полоса загрузки сверху в фирменном фиолетовом цвете
- Сохраняет вход (cookies/сессия)
- Ссылки на Telegram и VK открываются в их приложениях
- Скачивание файлов через системный загрузчик
- Иконка приложения в стиле сайта

## Как собрать APK — БЕЗ установки Android Studio

Самый простой путь — собрать проект онлайн.

### Вариант 1: онлайн-сборка (рекомендуется)

1. Заархивируй папку `AndroidApp` в ZIP (она уже в ZIP, если скачал архив).
2. Зайди на один из бесплатных сервисов онлайн-сборки APK, например:
   - **GitHub + GitHub Actions** (бесплатно, надёжно) — см. ниже
   - любой онлайн Gradle-сборщик
3. Загрузи проект, запусти сборку, скачай готовый `.apk`.

### Вариант 2: через GitHub Actions (бесплатно, полностью автоматически)

1. Создай аккаунт на github.com (если нет).
2. Создай новый репозиторий, загрузи туда всё содержимое папки `AndroidApp`.
3. Файл `.github/workflows/build.yml` уже внутри проекта — GitHub сам соберёт APK.
4. После загрузки: вкладка **Actions** → дождись зелёной галочки → внизу страницы
   в разделе **Artifacts** скачай `app-debug.apk`.
5. Перекинь APK на телефон и установи (разреши «установку из неизвестных источников»).

### Вариант 3: Android Studio (если передумаешь)

1. Открой папку `AndroidApp` в Android Studio.
2. Подожди, пока подтянутся зависимости.
3. Меню **Build → Build APK(s)**.
4. APK появится в `app/build/outputs/apk/`.

## Установка APK на телефон

1. Перекинь `.apk` на телефон (через кабель, Telegram себе, облако).
2. Открой файл на телефоне.
3. Android попросит разрешить установку из этого источника — разреши.
4. Установится приложение «KREZIBOY» с иконкой.

## Важно

- Приложение жёстко привязано к адресу `kolosoww70.temp.swtest.ru`.
  Если сайт переедет на другой домен — нужно поменять адрес в файле
  `app/src/main/java/ru/kreziboy/app/MainActivity.java`
  (строка `SITE_URL`) и пересобрать APK.
- Это **debug**-сборка — её достаточно для раздачи игрокам напрямую.
  Для публикации в Google Play нужна подписанная release-сборка
  и аккаунт разработчика Google (разовая оплата $25).

## Изменить название / адрес / иконку

- **Адрес сайта**: `app/src/main/java/ru/kreziboy/app/MainActivity.java` → `SITE_URL`
- **Название приложения**: `app/src/main/res/values/strings.xml` → `app_name`
- **Иконка**: замени файлы `ic_launcher.png` в папках `app/src/main/res/mipmap-*`
